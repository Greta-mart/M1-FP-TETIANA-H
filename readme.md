# Caesar Cipher

## Коротко про проєкт

Проєкт реалізує шифр Цезаря для роботи з текстовими файлами українською та англійською мовами. Програма підтримує шифрування, розшифрування та відновлення тексту методом brute-force.

## Що вдалося реалізувати

* Шифрування тексту за допомогою шифру Цезаря.
* Розшифрування тексту за відомим ключем.
* Роботу з файлами (читання та запис результатів у нові файли).
* Підтримку українського та англійського алфавітів.
* Автоматичне створення нових файлів для результатів шифрування та розшифрування.
* Brute-force режим для підбору ключа та відновлення оригінального тексту.
* Обробку граничних випадків та валідацію вхідних даних.

## Особливості проєкту

* Підтримуються дві мови: українська та англійська.
* Для кожної мови використовується власний алфавіт та словник частотних слів.
* Логіка проєкту розділена на окремі класи відповідно до їх відповідальності.
* Результати роботи зберігаються в окремих файлах без зміни оригінального файлу.

## Цікаві рішення

* Реалізовано автоматичне визначення найкращого ключа під час brute-force за допомогою оцінки якості тексту.
* Для пошуку правильного варіанта використовується аналіз частотних слів відповідної мови.
* Шифрування та розшифрування працюють через єдиний механізм зсуву символів.
